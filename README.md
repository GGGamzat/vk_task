# vk_task


Инструменты: Make, Telegram Bot API, PDF.co, Google Sheets<br>
Сложная часть: Извлечение текста из PDF и настройка цепочки данных<br>
Улучшения:<br>
  1. Добавить OpenAI для суммаризации и ключевых слов
  2. Добавить Router для обработки TXT/DOCX файлов
  3. Добавить OCR для сканов

Выявленная проблема: В процессе настройки столкнулся с технической проблемой интерфейса модуля OpenAI в Make.com. Независимо от выбранной модели (GPT-5 mini или другие) и корректности введённых переменных (например, {{4.Body}}), модуль при сохранении выдавал постоянную ошибку валидации подключения без детализации. При этом другие модули (Telegram, PDF.co, Google Sheets) с аналогичным принципом подключения работали штатно.
